== Installation ==

1. Install using the WordPress Pluin Installer (search for `Appapro iAthletics`) or download `Appapro iAthletics`, extract the `appapro-iathletics` folder and upload `appapro-iathletics` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure `Appapro iAthletics` from the `Settings` menu as appropriate.

== Usage ==

The Appapro iAthletics shortcode `aia` supports a single attribute.  The rest of the controls are derived from the information stored with the Custom Post Type.

`[aia id='<Appapro iAthletics Team Id>']`

*NOTE:*  In the above syntax, values enclosed in angle brackets <>, indicate a string you need to replace with an appropriate value.  Do not include the angle brackets in your string!

* __id__:  The numeric id of the Appapro iAthletics Team (Appapro will provide this).
